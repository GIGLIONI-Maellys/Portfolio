Daktari - Fini

Description du projet

Le projet Daktari est constitué de deux phases. Dans une première phase, l'objectif est la mise en place d'une base de données sur MySQL répondant aux besoins d'un vétérinaire fictif, Daktari. Dans une deuxième phase, les objectifs sont le passage de la base de données sous MySQL vers PostgreSQL puis la réalisation d'un site web sous HTML et CSS, interface pour la gestion de la base de données de Monsieur Daktari, où il peut stocker ses informations quant aux animaux, consultations, traitements, manipulations, etc...
