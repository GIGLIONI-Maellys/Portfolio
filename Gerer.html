SAE - Projet TableauLEGO

Pour la SAE de deuxième année, nous avons travaillé sur le projet TableauLEGO. Celui-ci consiste en la réalisation d'un site de e-commerce de création et de vente de tableaux de briques. Pour la deuxième partie de cette SAE, au semestre 4, nous avons également du réaliser un jeu type Tetris au format site web et android. Au cours de ce projet sur deux semestres :

  • Nous avons veillé à la conformité RGPD du projet tout au long, par exemple en isolant les données sensibles et en limitant les droits d'accès seuls aux personnes autorisées. De plus pour limiter les accès frauduleux aux comptes, nous avons utilisé une authentification à deux facteurs.
  • Afin que le code puisse être maintenu après la livraison, nous avons réalisé des ReadMe et constamment veillé à ce que notre code reste le plus propre possible, afin qu'il puisse être évolutif. Nous avons également tout au long du projet utilisé l'outil de versionning Git, afin de garder un historique sain du projet,
  • Une attention a également été portée aux requêtes effectuées à la base de données, celle-ci étant déjà de taille imposante, afin que l'expérience utilisateur n'en soit pas déteriorée et que le processus puisse fonctionner au mieux peut importe la taille de la base de données.
